Balloon Ballet
######################################

@date: 06/22/2017
@author: Robert J Paul

This is a fun little game to showcase my CANNON, THREE, and webGL abilities. Had a fun time doing it too!

To run with grunt, make sure grunt, watch, concat, uglify, and express are all installed

npm install grunt --save-dev
npm install grunt-contrib-watch --save-dev
npm install grunt-contrib-concat --save-dev
npm install grunt-contrib-uglify --save-dev
npm install grunt-express --save-dev

Then run 'grunt' 

Go to 'localhost:9000' to see it all in action
